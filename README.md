# class_repayment_calculator

Personal **Debt & Cashflow Visualizer** — track credit cards, medical debt, education clawback, income, Chime savings, and rolling projections.

## Deploy

Static site: open `index.html` locally, or deploy the repo root to [Netlify](https://www.netlify.com/) (publish directory `.`, no build command).

## Data

All data is stored in the browser (`localStorage`) per device. Use **Export JSON** / **Import JSON** to back up or move your data.
